Suiboku

Introduction

  • The name “Suiboku” is derived from East Asian ink wash painting.
  • base color
    WASHI
    traditional Japanese paper (light background)
    AOZUMI
    Suiboku/Shodo's ink (light foreground)
    Ash
    light sub foreground
    Inkstone
    Suiboku/Shodo's inkstone (dark background)
    Fog
    soft paper-like light (dark foreground)
    Gray
    dark sub foreground
  • inspire by Solarized.
    • When switching between light and dark modes, only the base color changes; other colors remain unchanged.
    • Support both light/dark mode.
  • Do not use italic, for default.
    • Because it became difficult to read when wide character are used.
      To use italic:
      let g:suiboku = #{italic: v:true, transparent: v:true}
  • In terminal, support transparency.
    • To use the transparency effect, you need to
      • use enable transparency terminal
      • set background color.
      the colorscheme set
      highlight Normal ctermbg=NONE guibg=NONE
    • To disable transparency:
      let g:suiboku = #{italic: v:false, transparent: v:false}
