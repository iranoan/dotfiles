Solarized9

Introduction

  • Solarized base
  • Use invert 'set background=light/dark' background color to make the default (Normal) text color.
  • Do not use italic, for default.
    • Because it became difficult to read when wide character are used.
      To use italic:
      let g:solarized9 = #{italic: v:true, transparent: v:true}
  • In terminal, support transparency.
    • To use the transparency effect, you need to
      • use enable transparency terminal
      • set the Solarized theme in the terminal. This is because it uses the terminal's background color.
      Solarize9 set
      highlight Normal ctermbg=NONE guibg=NONE
    • To disable transparency:
      let g:solarized9 = #{italic: v:false, transparent: v:false}

Setting order

  1. Normal
  2. highlight-default OR group-name (not a link)
    • not italic or italic
  3. other (not a link)
  4. highlight-default OR group-name (link)
  5. other (link)
  • 1-3: Order is dark > light > common.
  • 1-3: Sort alphabetically for each category: dark, light and common.
    • Prioritize highlight-default and 'suggested group names'.
  • 4-5: Sort by link destination.
    • Prioritize highlight-default and 'suggested group names'.
